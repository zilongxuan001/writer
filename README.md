### 周报写作流程

（1）在【Issues】建立“issue01”,为第一周，将每日有触动的信息放在isuue01里。
（2）到周五时，在【docs/】里建立“issues01.md”，整理【Issues】里【issue01】的信息为文章。
（3）发表在自己的其他账号里，比如简书，知乎等等。
